# PROYECTO SOFTWARE TIPO PREDIK

Plataforma propia de inteligencia geoespacial y geomarketing, inspirada funcionalmente en GeoData Intelligence (PREDIK Data-Driven), construida con fuentes de datos propias (INEGI y otras APIs configurables).

## Estructura del repositorio

```
.
├── backend/      → API (FastAPI) + conectores de datos + base de datos
├── frontend/      → Aplicación web (React)
├── docs/           → Documentación técnica completa (arquitectura, APIs, BD, ADRs, etc.)
├── infra/           → Docker Compose y configuración de infraestructura
└── .github/          → Pipelines de CI/CD
```

## Por dónde empezar

1. Lee primero `docs/00-README.md` — es el índice de toda la documentación técnica.
2. Para levantar el ambiente local, ver instrucciones en `infra/` (próximamente).
